feat(iam): 实现 WebAuthn 无密码登录和硬件密钥支持

## 新增功能

### 1. WebAuthn 集成
- 集成 webauthn-rs (v0.5) 库
- 支持 FIDO2/WebAuthn 标准
- 兼容硬件密钥和平台认证器

### 2. 数据库层
- 新增 webauthn_credentials 表
- 存储凭证 ID、公钥、计数器等信息
- 支持多种传输方式（USB, NFC, BLE, Internal）
- 添加索引优化查询性能

### 3. 领域层
- 新增 WebAuthnCredential 实体
  - 完整的凭证生命周期管理
  - 签名计数器防重放攻击
  - 备份状态跟踪
- 新增 WebAuthnCredentialRepository trait
  - 标准 CRUD 操作
  - 按用户查询凭证
  - 凭证所有权验证
- 新增 WebAuthnService 领域服务
  - 注册流程管理
  - 认证流程管理
  - 凭证管理功能

### 4. 基础设施层
- PostgresWebAuthnCredentialRepository 实现
  - 完整的仓储接口实现
  - 二进制数据正确处理
  - 完善的错误处理

### 5. API 层
- 新增 6 个 gRPC 接口
  - StartWebAuthnRegistration - 开始注册
  - FinishWebAuthnRegistration - 完成注册
  - StartWebAuthnAuthentication - 开始认证
  - FinishWebAuthnAuthentication - 完成认证
  - ListWebAuthnCredentials - 列出凭证
  - DeleteWebAuthnCredential - 删除凭证
- Proto 消息定义完善
- 完整的请求/响应处理

### 6. 服务集成
- main.rs 集成 WebAuthn 服务
- 配置 Relying Party ID 和 Origin
- 依赖注入到 AuthService
